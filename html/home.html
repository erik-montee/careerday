I built this website to help me demonstrate to you what I do on a daily basis and to introduce you to the basics concepts of programming. There are many resources out there to help you learn to code in an interactive way. One of the best resources to learn is scratch. scratch is an interactive learning GUI for programming created by MIT to introduce children to programming.

The graphic to the right was made completely in scratch. You can find more examples at scratchs website https://scratch.mit.edu
