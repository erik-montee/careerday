Markup Languages

Markup languages is used to label individual parts of the document and help create a discription of the data. The common markup languages used in web development are HTML and XML. HTML is what your browser reads to know what type of tag is being used.

The <b> makes the text bold

The <i> makes the text italics.

Style Sheets

What most people are use to looking at is how the website is displayed on the page. This is done via CSS. This tells your browser how to display certain elements on the page. They can be updated in the browser.


Client Side Languages

These are programming languages that run inside your browser and your computer is doing the processing. You are not gathering information from any other location(usually). The main one is usually javascript.

Server Side languages

These are programming languages that are run on the computer that is sending your computer information. As a user of a website you never run into these languages. These languages are used to do tasks and then send the information to the user
